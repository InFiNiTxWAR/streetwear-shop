# 🛍 NovaStore

Site de e-commerce desenvolvido para exibir produtos eletrônicos e roupas utilizando **HTML, CSS e JavaScript**, consumindo dados de uma **API de produtos**.

O objetivo do projeto é simular uma loja online moderna com design escuro, animações em CSS e carregamento dinâmico de produtos.

---

## 📌 Funcionalidades

* Página inicial com **hero moderno**
* Seção **Sobre a loja**
* **Carrossel de produtos em destaque**
* **Grid de produtos carregados via API**
* Cards de produtos com imagem, nome, categoria e preço
* Layout **responsivo**
* Design **dark mode com gradientes e animações**

---

## 🔌 API utilizada

O projeto utiliza a API pública:

https://fakestoreapi.com/products

Ela fornece produtos com:

* nome
* preço
* imagem
* categoria
* descrição

Esses dados são carregados dinamicamente no site usando JavaScript.
